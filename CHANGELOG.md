# What's New in OrbyGlasses

## Version 0.9.0 (Current)

**Release Date:** November 2025

### What Changed

**Better Safety:**
- System now stops automatically if something is too close (<0.5 meters)
- Press spacebar or 'q' anytime for emergency stop
- 5 different safety checks running at once
- If one check fails, others keep you safe

**Faster:**
- Audio warnings now come in 0.5 seconds (used to be 2 seconds)
- Works 5x faster on M1/M2/M3/M4 Macs
- Computer automatically uses fastest mode available

**Easier to Use:**
- Simpler setup - just run 2 commands
- Better error messages that make sense
- Automatic recovery if something stops working

**Removed:**
- 22 old test files nobody was using
- 15 old setup scripts
- 108MB of extra code

### Bug Fixes
- Fixed camera tracking in normal rooms
- Fixed audio delay problems
- Fixed memory leaks
